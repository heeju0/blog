TIL - 2021.04.24

April 24, 2021 · 1 min read

Heeju Lee

Dictionary객체.get(key값, 기본값): 딕셔너리 타입의 데이터에서 key로 value 값을 얻고, 해당 key가 없을 경우 설정한 기본값을 리턴한다.

리스트를 딕셔너리 형태로 데이터를 바꾸고자 할 때, 이 메서드를 사용하여 보다 간단하게 구현할 수 있다.

기본
dic = {}
for k, v in lst:
if k in dic:
dic[k].append(v)
else:
dic[k] = [v]
get() 메서드 사용
dic = {}
for k, v in lst:
dic[k] = dic.get(k, []) + [v]
  • 기타 방법
defaultdict 사용
from collections import defaultdict
dic = defaultdict(list)
for k, v in lst:
dic[k].append(v)
