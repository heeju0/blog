TIL2 - Day3

April 24, 2021 · 1 min read

Heeju Lee

Dictionary객체.get(key값, 기본값): 딕셔너리 타입의 데이터에서 key로 value 값을 얻고, 해당 key가 없을 경우 설정한 기본값을 리턴한다.

리스트를 딕셔너리 형태로 데이터를 바꾸고자 할 때, 이 메서드를 사용하여 보다 간단하게 구현할 수 있다.

기본
dic = {}
for k, v in lst:
if k in dic:
dic[k].append(v)
else:
dic[k] = [v]
get() 메서드 사용
dic = {}
for k, v in lst:
dic[k] = dic.get(k, []) + [v]
  • 기타 방법
defaultdict 사용
from collections import defaultdict
dic = defaultdict(list)
for k, v in lst:
dic[k].append(v)

TIL1 - Day1

April 23, 2021 · 1 min read

Heeju Lee

itertools 라이브러리를 이용하여 순열과 조합을 쉽게 계산할 수 있다.

itertools.permutations(배열, r): 배열 내 서로 다른 n개의 값 중에서 r개를 선택하는 모든 경우의 수 (ₙPᵣ)
itertools.combinations(배열, r): 배열 내 서로 다른 n개의 값 중에서 순서에 상관 없이 r개를 선택하는 모든 경우의 수 (ₙCᵣ)

Day 1의 Lv.2 사탕 담기 문제의 경우 전부 구현하려 하면 복잡하지만 이 combinations 메서드를 이용하면 간단하게 풀 수 있었다. (역시 아는 게 힘인가 보다..)
