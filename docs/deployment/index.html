Deployment

💡 Docusaurus v2 문서 참고
💡 Visual Studio Code 사용 시 편리

Github에 저장 및 배포#

설정#

docusaurus.config.js > <unknown> 에서 작업

module.exports = {
...,
url: 'https://아이디.github.io',
baseUrl: '/레포명/',
...,
organizationName: '아이디',
projectName: '레포명',
...
}

저장#

git add .
git commit -m "first commit"
git push -u origin master

배포#

Ctrl + C로 로컬 서버를 종료 후 실행

#Bash
GIT_USER=<GITHUB_USERNAME> yarn deploy
#Windows
cmd /C "set "GIT_USER=<GITHUB_USERNAME>" && yarn deploy"
#PowerShell
cmd /C 'set "GIT_USER=<GITHUB_USERNAME>" && yarn deploy'

블로그 확인#

https://아이디.github.io/레포명/
