Getting Started

💡 Docusaurus v2 문서 참고
💡 Visual Studio Code 사용 시 편리

Github Repo 생성 및 연결#

Repository 생성#

https://github.com/new에서 Repository name을 정한 뒤 생성

Desktop과 연결#

❗ cmd 창에서 진행

cd [폴더 경로]
git config --global user.name "[아이디]"
git config --global user.email "[이메일]"
git init
git add .
git commit -m "first commit"
git remote add origin https://github.com/[아이디]/[레포명].git
git push -u origin master

설치 및 개발 서버 실행#

node.js 설치
yarn 설치

npx @docusaurus/init@latest init my-website classic
cd my-website
yarn start 혹은 npm run start
