내 풀이#

❗ 여벌 체육복을 가져온 학생이 도난당한 경우, 빌려줄 수 없다는 점을 간과했다.

def solution(n, lost, reserve):
for i in range(1, n+1):
if i in lost and i in reserve:
lost.remove(i)
reserve.remove(i)
answer = n - len(lost)
for i in reserve:
if i-1 in lost:
answer += 1
lost.remove(i-1)
elif i + 1 in lost:
answer += 1
lost.remove(i+1)
return answer

내 풀이#

❗ 무작정 최댓값 찾아서 하려고 하면 한 문제 오류 + 두 문제 시간초과 뜬다 ㅠㅠ

def solution(number, k):
lst = [number[0]]
for i in number[1:]:
while len(lst) > 0 and lst[-1] < i and k > 0:
k -= 1
lst.pop()
lst.append(i)
if k != 0:
lst = lst[:-k]
answer = ''.join(lst)
return answer
