1. Matplotlib - 기본 그래프 도구#

Matplotlib은 2D 평면 그래프에 관한 다양한 포맷과 기능을 지원한다. 객체지향 프로그래밍을 지원하므로 그래프 요소를 세세하게 꾸밀 수 있다.

import matplotlib.pyplot as plt

Matplotlib 함수 사전#

✔ plt.plot() : 그래프#

kind='line': 선 그래프
marker: 점의 모양
ㅤ ㅤㅤ 'o', '+', '*', '.'
markerfacecolor: 점의 배경색
markersize: 점의 크기
color: 선 색
linewidth: 선 두께
label: 라벨 지정

kind='area': 면적 그래프
stacked: 누적 여부
alpah: 투명도

kind='bar'/'barh': 막대 그래프

kind='hist': 히스토그램
bins: 구간 수

kind='scatter': 산점도
c: 점의 색상
s: 점의 크기
marker: 점 모양
cmap: 컬러맵, 레퍼런스 참조

✔ plt.title() : 차트 제목#

size: 폰트 크기

✔ plt.xlabel(), plt.ylabel() : x, y축 이름#

size: 폰트 크기

✔ plt.show() : 변경사항 저장 및 그래프 출력#
✔ plt.figure() : 그림 사이즈 지정#

figsize: (가로 길이, 세로 길이)

✔ plt.xticks(), plt.yticks() : x, y축 눈금 설정#

rotation: 눈금 라벨 회전
ㅤㅤㅤㅤ'vertical'(수직), 0~360(각도)
size: 폰트 크기

✔ plt.style.use() : 스타일 서식 지정#

'classic', 'ggplot', 'bmh', 'dark_background', 'fast', 'grayscale', 'seaborn' 등
스타일 목록: plt.style.available 적용 결과에 대한 정보는 레퍼런스 참고

✔ plt.legend() : 범례 표시#

loc: 범례 위치
ㅤㅤ'best'(추천)

✔ plt.xlim(), plt.ylim() : x, y축 범위 지정#

최솟값, 최댓값

✔ plt.annotate() : 주석 설정#

[화살표]
xy: 화살표 머리 부분(끝점)
xytext: 화살표 꼬리 부분(시작점)
xycoords: 좌표체계
arrowprops: 화살표 서식
ㅤㅤㅤㅤ ㅤ dict(arrowstyle='->', color='색깔', lw=길이)
[텍스트]
'텍스트 내용'
xy: 텍스트 위치 기준점
rotation: 텍스트 회전 각도(반시계방향)
va: 상하 정렬
ㅤㅤ'center', 'top', 'bottom', 'baseline'
ha: 좌우 정렬
ㅤㅤ'center', 'left', 'right'
fontsize: 폰트 크기

✔ fig.add_subplot() : 그림틀 분할#

행의 크기, 열의 크기, 서브플롯 순서

✔ ax.set_xlim(), ax.set_ylim() : axe 객체의 x, y축 범위 지정#

최솟값, 최댓값

✔ ax.set_xticklabels(), ax.set_xticklabels() : axe 객체의 x, y축 눈금 설정#

rotation: 눈금 라벨 회전
ㅤㅤㅤㅤ'vertical'(수직), 0~360(각도)
size: 폰트 크기

✔ ax.set_title() : axe 객체의 차트 제목#

size: 폰트 크기

✔ ax.set_xlabel(), ax.set_ylabel() : axe 객체의 x, y축 이름#

size: 폰트 크기

✔ ax.tick_params() : axe 객체의 x, y축 눈금 설정#

axis: 축 설정
ㅤㅤ 'x', 'y'
labelsize: 폰트 크기

✔ plt.savefig(): 그림 파일로 저장#

경로 입력
transparent: 배경 투명 여부

◾ 화면 분할하여 그래프 여러 개 그리기#

여러 개의 axe 객체를 만들고, 분할된 화면마다 axe 객체를 하나씩 배정한다. axe 객체는 각각 서로 다른 그래프를 표현할 수 있다.

figure() 함수를 사용하여 그림틀(fig)을 만들고, figsize 옵션으로 크기를 설정한다. fig 객체에 add_subplot() 메소드를 적용하여 그림틀을 여러 개로 분할하는데, 이때 나눠진 각 부분을 axe 객체라고 부른다.

note

fig.add_subplot(행의 크기, 열의 크기, 서브플롯 순서)

화면 분할하기
# 그래프 객체 생성(figure에 2개의 서브 플롯 생성)
fig = plt.figure(figsize=(10, 10))
ax1 = fig.add_subplot(2, 1, 1)
ax2 = fig.add_subplot(2, 1, 2)
# axe 객체에 plot 함수로 그래프 출력
ax1.plot(sr_one, 'o', markersize=10)
ax2.plot(sr_one, marker='o', markerfacecolor='green', markersize=10,
color='olive', linewidth=2, label='서울 -> 경기')
ax2.legend(loc='best')
# y축 범위 지정
ax1.set_ylim(50000, 800000)
ax2.set_ylim(50000, 800000)
# 축 눈금 라벨 지정 및 75도 회전
ax1.set_xticklabels(sr_one.index, rotation=75)
ax2.set_xticklabels(sr_one.index, rotation=75)
plt.show()

동일한 그림(axe 객체)에 여러 개의 그래프를 추가하는 것도 가능하다.

같은 화면에 그래프 추가
# 서울에서 '충청남도', '경상북도', '강원도'로 이동한 인구 데이터 값만 선택
col_years = list(map(str, range(1970, 2018)))

+ 보조 축 활용하기#

📚 마이너스 부호 출력 설정: plt.rcParams['axes.unicode_minus'] = False
📚 데이터를 n행씩 뒤로 이동: Series객체.shift(n)
📚 ax객체의 쌍둥이 객체 생성: ax객체.twinx()

2축 그래프 그리기
print(df)
# 발전 전력별 총발전량 수력 화력 원자력 총발전량 - 1년 증감률
# 1990 277 156 121 - NaN NaN
# 1991 263 150 113 - 277 -5.05415
# ... ... ... ... ... ... ...
# 2015 190 100 90 - 216 -12.037
# 2016 239 128 111 - 190 25.7895
plt.rcParams['axes.unicode_minus'] = False
ax1 = df[['수력', '화력']].plot(kind='bar', figsize=(20, 10), width=0.7, stacked=True)
ax2 = ax1.twinx()
ax2.plot(df.index, df.증감률, ls='--', marker='o', markersize=20,
color='red', label='전년대비 증감률(%)')
ax1.set_ylim(0, 500)
ax2.set_ylim(-50, 50)
ax1.set_xlabel('연도', size=20)
ax1.set_ylabel('발전량(억 KWh)')
ax2.set_ylabel('전년 대비 증감률(%)')
plt.title('북한 전력 발전량 (1990~2016)', size=30)
ax1.legend(loc='upper left')
plt.show()

1-7. 박스 플롯#

박스 플롯은 범주형 데이터의 분포를 파악하는데 적합하며, 5개의 통계 지표(최솟값, 1분위값, 중간값, 3분위값, 최댓값)를 제공한다.

박스 플롯
print(df)
# mpg cylinders displacement ... origin name
# 0 18.0 8 307.0 ... 1 chevrolet chevelle malibu
# 1 15.0 8 350.0 ... 1 buick skylark 320
# .. ... ... ... ... ... ...
# 397 31.0 4 119.0 ... 1 chevy s-10
fig = plt.figure(figsize=(15, 5))
ax1 = fig.add_subplot(1, 2, 1)
ax2 = fig.add_subplot(1, 2, 2)

ax2.boxplot(x=[df[df['origin'] == 1]['mpg'],
df[df['origin'] == 2]['mpg'],
df[df['origin'] == 3]['mpg']],
labels=['USA', 'EU', 'JAPAN'], vert=False)
