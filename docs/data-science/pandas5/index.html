2. Seaborn - 고급 그래프 도구#

Seaborn은 Matplotlib의 기능과 스타일을 확장한 파이썬 시각화 도구의 고급 버전이다.

import seaborn as sns

Seaborn 함수 사전#

✔ sns.set_palette() : 그래프에 사용할 팔레트 색 설정#

'deep', 'muted', 'pastel', 'bright', 'dark', 'colorbind'

✔ sns.set_style() : 스타일 테마 설정#

'darkgrid', 'whitegrid', 'dark', 'white', 'ticks'

✔ 그래프 공통 옵션#

x, y, hue
data: 입력 데이터
ax: axe 객체명
dodge: hue 옵션 설정 시 막대 그래프 분산 여부 (True/False)
palette: 그래프 색 구성
ㅤ ㅤㅤ 'Set1', 'Set2', 'Set3' 등

✔ sns.regplot() : 산점도와 회귀선 그래프#

sns.regplot(x, y, data=, ax=)
fit_reg: 회귀선 표시 여부 (True/False)

✔ sns.distplot() : 히스토그램/커널 밀도 그래프#

sns.distplot(x, ax=) hist: 히스토그램 표시 여부 (True/False) kde: 커널 밀도 그래프 표시 여부 (True/False)

✔ sns.heatmap() : 히트맵#

sns.heatmap(df)
annot: 데이터 값 표시 여부 (True/False)
fmt='d': 정수형 포맷
cmap: 컬러 맵
linewidth: 구분 선 두께
cbar: 컬러 바 표시 여부 (True/False)

✔ sns.stripplot() : 범주형 데이터의 산점도 그래프#

sns.stripplot(x, y, hue=, data=, ax=)

✔ sns.swarmplot() : 분산을 고려한 범주형 데이터의 산점도 그래프#

sns.swarmplot(x, y, hue=, data=, ax=)

✔ sns.barplot() : 막대 그래프#

sns.barplot(x, y, hue=, data=, ax=)

✔ sns.boxplot() : 박스 플롯#

sns.boxplot(x, y, hue=, data=, ax=)

✔ sns.violinplot() : 바이올린 플롯#

sns.violinplot(x, y, hue=, data=, ax=)

✔ sns.jointplot() : 조인트 그래프#

sns.jointplot(x, y, data=, ax=) kind: 기본값(산점도), 'reg'(회귀선), 'hex'(육각 그래프), 'kde'(커널 밀도 그래프)

✔ sns.FacetGrid() : 조건에 맞게 화면 분할#

sns.jointplot(data=, col=, row=)

✔ sns.pairplot() : 두 개씩 짝지을 수 있는 모든 조합 그래프#

2-1. 회귀선이 있는 산점도#

❗ Seaborn 라이브러리에서 제공하는 titanic 데이터 셋을 사용한다.

회귀선이 있는 산점도
titanic = sns.load_dataset('titanic')
sns.set_style('darkgrid')
fig = plt.figure(figsize=(12, 5))
ax1 = fig.add_subplot(1, 2, 1)
ax2 = fig.add_subplot(1, 2, 2)
sns.regplot('age', 'fare', data=titanic, ax=ax1, color='darkred')
sns.regplot('age', 'fare', data=titanic, ax=ax2, fit_reg=False)
plt.show()

2-2. 히스토그램/커널 밀도 그래프#

히스토그램/커널밀도함수
fig = plt.figure(figsize=(15, 5))
ax1 = fig.add_subplot(1, 3, 1)
ax2 = fig.add_subplot(1, 3, 2)
ax3 = fig.add_subplot(1, 3, 3)
sns.distplot(titanic['fare'], ax=ax1)
sns.distplot(titanic['fare'], hist=False, ax=ax2)
sns.distplot(titanic['fare'], kde=False, ax=ax3)
ax1.set_title('titanic fare = hist/kde')
ax2.set_title('titanic fare = kde')
ax3.set_title('titanic fare = hist')
plt.show()

타이타닉호의 운임은 대부분 100달러 미만에 분포하고 있다.

2-3. 히트맵#

히트맵
table = titanic.pivot_table(index='sex', columns='class', aggfunc='size')
sns.heatmap(table, annot=True, fmt='d', cmap='YlGnBu', linewidth=.5, cbar=False)
plt.show()

여자 승객보다 남자 승객이 더 많으며, 3등석 남자 승객의 수가 압도적으로 많다.

2-4. 범주형 데이터의 산점도#

범주형 데이터의 산점도
fig = plt.figure(figsize=(15, 5))
ax1 = fig.add_subplot(1, 2, 1)
ax2 = fig.add_subplot(1, 2, 2)
sns.stripplot('class', 'age', data=titanic, ax=ax1)
sns.swarmplot('class', 'age', data=titanic, ax=ax2)
ax1.set_title('Strip Plot')
ax2.set_title('Swarm Plot')
plt.show()

1등석 승객의 연령대가 높은 편이다.

fig = plt.figure(figsize=(15, 5))
ax1 = fig.add_subplot(1, 2, 1)
ax2 = fig.add_subplot(1, 2, 2)
sns.stripplot('class', 'age', hue='sex', data=titanic, ax=ax1)
sns.swarmplot('class', 'age', hue='sex', data=titanic, ax=ax2)
ax1.set_title('Strip Plot')
ax2.set_title('Swarm Plot')
plt.show()

2-5. 막대 그래프#

막대 그래프
fig = plt.figure(figsize=(15, 5))
ax1 = fig.add_subplot(1, 3, 1)
ax2 = fig.add_subplot(1, 3, 2)
ax3 = fig.add_subplot(1, 3, 3)
sns.barplot('sex', 'survived', data=titanic, ax=ax1)
sns.barplot('sex', 'survived', hue='class', data=titanic, ax=ax2)
sns.barplot('sex', 'survived', hue='class', dodge=False, data=titanic, ax=ax3)
ax1.set_title('titanic survived - sex')
ax2.set_title('titanic survived - sex/class')
ax3.set_title('titanic survived - sex/class(stacked)')
plt.show()

남자 승객보다 여자 승객의 생존률이 훨씬 높으며, 좌석 등급이 높을수록 생존률이 높아진다.

2-6. 빈도 그래프#

빈도 그래프
fig = plt.figure(figsize=(15, 5))
ax1 = fig.add_subplot(1, 3, 1)
ax2 = fig.add_subplot(1, 3, 2)
ax3 = fig.add_subplot(1, 3, 3)
sns.countplot('class', palette='Set1', data=titanic, ax=ax1)
sns.countplot('class', hue='who', palette='Set2', data=titanic, ax=ax2)
sns.countplot('class', hue='who', palette='Set3', dodge=False, data=titanic, ax=ax3)
ax1.set_title('titanic class')
ax2.set_title('titanic class - who')
ax3.set_title('titanic class - who(stacked)')
plt.show()

2-7. 박스 플롯/바이올린 그래프#

박스 플롯/바이올린 그래프
sns.set_style('whitegrid')
fig = plt.figure(figsize=(15, 10))
ax1 = fig.add_subplot(2, 2, 1)
ax2 = fig.add_subplot(2, 2, 2)
ax3 = fig.add_subplot(2, 2, 3)
ax4 = fig.add_subplot(2, 2, 4)
sns.boxplot('alive', 'age', data=titanic, ax=ax1)
sns.boxplot('alive', 'age', hue='sex', data=titanic, ax=ax2)
sns.violinplot('alive', 'age', data=titanic, ax=ax3)
sns.violinplot('alive', 'age', hue='sex', data=titanic, ax=ax4)
plt.show()

2-8. 조인트 그래프#

조인트 그래프
j1 = sns.jointplot('fare', 'age', data=titanic)
j2 = sns.jointplot('fare', 'age', kind='reg', data=titanic)
j3 = sns.jointplot('fare', 'age', kind='hex', data=titanic)
j4 = sns.jointplot('fare', 'age', kind='kde', data=titanic)
j1.fig.suptitle('titanic fare - scatter', size=15)
j2.fig.suptitle('titanic fare - reg', size=15)
j3.fig.suptitle('titanic fare - hex', size=15)
j4.fig.suptitle('titanic fare - kde', size=15)
plt.show()

조건에 맞게 화면 분할

g = g.map(plt.hist, 'age')
