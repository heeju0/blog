1. 데이터프레임의 구조#

❗ UCI 머신러닝 저장소에서 제공하는 자동차 연비(auth mpg) 데이터셋을 사용한다.

No.attributes속성데이터 상세(범위)
1mpgㅤㅤㅤㅤㅤ연비연속형
2cylindersㅤㅤㅤ실린더 수이산형
3displacementㅤ배기량연속형
4horsepowerㅤ출력연속형
5weightㅤㅤㅤ차중연속형
6accelerationㅤ가속능력연속형
7model_yearㅤ출시년도이산형
8originㅤㅤㅤㅤ제조국이산형
9nameㅤㅤㅤㅤ모델명문자열

2. 통계 함수 적용#

2-1. 평균값#

📚 모든 열의 평균값: DataFrame객체.mean()
📚 특정 열의 평균값: DataFrame객체['열 이름'].mean()

통계 함수
print(df.mean())
# mpg 23.514573
# cylinders 5.454774
# displacement 193.425879
# weight 2970.424623
# acceleration 15.568090
# model year 76.010050
# origin 1.572864
# dtype: float64
print(df['mpg'].mean())
# 23.514572864321615
print(df.mpg.mean())
# 23.514572864321615
print(df[['mpg', 'weight']].mean())
# mpg 23.514573
# weight 2970.424623
# dtype: float64

2-2. 중간값#

📚 모든 열의 중간값: DataFrame객체.median()
📚 특정 열의 중간값: DataFrame객체['열 이름'].median()

print(df.median())
# mpg 23.0
# cylinders 4.0
# displacement 148.5
# weight 2803.5
# acceleration 15.5
# model year 76.0
# origin 1.0
# dtype: float64
print(df['mpg'].median())
# 23.0

3. 판다스 내장 그래프 도구 활용#

📚 그래프: DataFrame객체.plot()

kind 옵션설명
'line'선 그래프
'bar'수직 막대 그래프
'barh'수평 막대 그래프
'his'히스토그램
'box'박스 플롯
'kde'커널 밀도 그래프
'area'면적 그래프
'pie'파이 그래프
'scatter'산점도 그래프
'hexbin'고밀도 산점도 그래프

[표 3-2] 판다스 내장 plot() 메소드 - 그래프 종류

  • 선 그래프
선 그래프 그리기
df_ns = df.iloc[[0, 5], 3:]
df_ns.index = ['South', 'North'] # 합계 데이터만 추출
df_ns.columns = df_ns.columns.map(int) # 자료형을 정수형으로 변경
print(df_ns.head(3))
# 1991 1992 1993 1994 1995 1996 ... 2014 2015 2016
# South 1186 1310 1444 1650 1847 2055 ... 5220 5281 5404
# North 263 247 221 231 230 213 ... 216 190 239
df_ns.plot()

tdf_ns = df_ns.T
print(tdf_ns.head(3))
# South North
# 1991 1186 263
# 1992 1310 247
# 1993 1444 221
tdf_ns.plot()

  • 막대 그래프
막대 그래프
tdf_ns.plot(kind='bar')

  • 히스토그램
히스토그램
tdf_ns.plot(kind='hist')

  • 산점도
산점도
df = pd.read_csv('./auto-mpg.csv', header=None)
df.columns = ['mpg', 'cylinders', 'displacement', 'horsepower', 'weight',
'acceleration', 'model year', 'origin', 'name']
df.plot(x='weight', y='mpg', kind='scatter')

  • 박스 플롯
박스 플롯
df[['mpg', 'cylinders']].plot(kind='box')
