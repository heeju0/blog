3. Folium - 지도 활용#

Folium 라이브러리는 지도 위에 시각화할 때 유용한 도구이다.

Folium 설치
conda install -c conda-forge folium
import folium

Folium 함수 사전#

✔ folium.Map() : 지도 시각화#

location=[위도, 경도]: 해당 지점의 지도
zoom_start: 화면 확대 비율 조절
tiles: 지도 스타일 변경
ㅤ ㅤ'Stamen Terrain', 'Stamen Toner' 등

✔ folium.Marker(): 마커 표시#

folium.Marker([lat, lng])
popup: 마커를 클릭했을 때 팝업창에 나타낼 텍스트

✔ folium 함수.add_to(folium 객체) : folium 객체에 함수 결과 저장#
✔ folium.CircleMarker() : 원형 마커 표시#

folium.CircleMarker([lat, lng])
radius: 원의 반지름
color: 원의 둘레 색상
fill: 원 내부 색칠 여부
fill_color: 원을 채우는 색
fill_opacity: 투명도
popup: 마커를 클릭했을 때 팝업창에 나타낼 텍스트

✔ folium.Choropleth : 단계구분도 표시#

folium.Choropleth(geo_data=, data=, columns=)
geo_data: 지도 경계
data: 표시하려는 데이터
fill_color: 영역을 채우는 색
fill_opacity: 영역 투명도
line_opacity: 영역 둘레 투명도
threshold_scale: 단계 설정
key_on: 각 영역명을 나타내는 속성 지정
ㅤ ㅤㅤ 'feature.properties.name' 등

3-1. 지도 스타일 적용하기#

Folium은 웹 기반 지도를 만들기 때문에 웹 환경에서만 지도를 확인할 수 있다.

  • Spyter: save() 메소드를 적용하여 HTML 파일로 저장한 뒤 웹 브라우저에서 파일을 열어 확인
  • 웹 기반 IDE는 사용 가능 (Jupyter lab 등)
지도 스타일 적용
seoul_map1 = folium.Map(location=[37.55, 126.98], tiles='Stamen Terrain', zoom_start=12)
seoul_map2 = folium.Map(location=[37.55, 126.98], tiles='Stamen Toner', zoom_start=15)

3-2. 지도에 마커 표시하기#

❗ 서울지역 대학교 위치 데이터 셋을 사용한다.

지도에 마커 표시
seoul_map = folium.Map(location=[37.55, 126.98], tiles='Stamen Terrain', zoom_start=12)
for name, lat, lng in zip(df.index, df['위도'], df['경도']):
folium.Marker([lat, lng], popup=name).add_to(seoul_map)

지도에 원형 마커 표시
seoul_map = folium.Map(location=[37.55, 126.98], tiles='Stamen Terrain', zoom_start=12)
for name, lat, lng in zip(df.index, df['위도'], df['경도']):
folium.CircleMarker(
[lat, lng], radius=10, color='brown', fill=True,
fill_color='coral', fill_opacity=0.7, popup=name
).add_to(seoul_map)

3-3. 지도 영역에 단계구분도(Choropleth Map) 표시하기#

지도 영역에 단계구분도 표시하기
g_map = folium.Map(location=[37.5502, 126.982], tiles='Stamen Terrain', zoom_start=9)
year = '2007'
folium.Choropleth(
geo_data=geo_data, data=df[year], columns=[df.index, df[year]],
fill_color='YlOrRd', fill_opacity=0.7, line_opacity=0.3,
threshold_scale=[10000, 100000, 300000, 500000, 700000],
key_on='feature.properties.name'
).add_to(g_map)
