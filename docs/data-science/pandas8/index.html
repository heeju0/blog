데이터프레임 응용

💡 책 『파이썬 머신러닝 판다스 데이터 분석』으로 공부중..👀

1. 함수 매핑#

함수 매핑은 시리즈 또는 데이터프레임의 개별 원소를 특정 함수에 일대일 대응시키는 과정을 의미한다.

1-1. 개별 원소에 함수 매핑#

❗ Seaborn 라이브러리에서 제공하는 titanic 데이터 셋을 사용한다.

📚 시리즈 원소에 함수 매핑: Series객체.apply(매핑 함수)

시리즈의 원소에 apply() 적용
def add_10(n):
return n + 10
print(df.head(3))
# age fare
# 0 22.0 7.2500
# 1 38.0 71.2833
# 2 26.0 7.9250
print(df['age'].apply(add_10).head(3))
# 0 32.0
# 1 48.0
# 2 36.0
# Name: age, dtype: float64

📚 데이터프레임 원소에 함수 매핑: DataFrame객체.applymap(매핑 함수)

데이터프레임 원소에 applymap() 적용
print(df.applymap(add_10).head(3))
# age fare
# 0 32.0 17.2500
# 1 48.0 81.2833
# 2 36.0 17.9250

1-2. 시리즈 객체에 함수 매핑#

📚 데이터프레임의 열에 함수 매핑: DataFrame객체.apply(매핑 함수, axis=0)

시리즈를 입력받고 시리즈를 반환하는 함수를 매핑하면 데이터프레임을 반환하고, 시리즈를 입력받아 하나의 값을 반환하는 함수를 매핑하면 시리즈를 반환한다.

데이터프레임에 apply(axis=0) 적용
def missing_value(series):
return series.isnull()
print(df.apply(missing_value, axis=0).head(3))
# age fare
# 0 False False
# 1 False False
# 2 False False
def min_max(x):
return x.max() - x.min()
print(df.apply(min_max))
# age 79.5800
# fare 512.3292
# dtype: float64
데이터프레임에 apply(axis=1) 적용
def add_two_obj(a, b):
return a + b
print(df.apply(lambda x: add_two_obj(x['age'], 10), axis=1).head(3))
# 0 32.0
# 1 48.0
# 2 36.0
# dtype: float64

1-3. 데이터프레임 객체에 함수 매핑#

📚 데이터프레임 객체에 함수 매핑: DataFrame객체.pipe(매핑 함수)

데이터프레임 반환, 시리즈 반환, 개별 값 반환이 가능하다.

데이터프레임에 pipe() 적용
def missing_value(x):
return x.isnull()
def missing_count(x):
return missing_value(x).sum()
def total_number_missing(x):
return missing_count(x).sum()
result_df = df.pipe(missing_value)
print(result_df.head(3))
# age fare
# 0 False False
# 1 False False
# 2 False False
result_series = df.pipe(missing_count)
print(result_series)
# age 177
# fare 0
# dtype: int64
result_value = df.pipe(total_number_missing)
print(result_value)
# 177

2. 열 재구성#

2-1. 열 순서 변경#

📚 데이터프레임의 열 순서 변경: DataFrame객체[재구성한 열 이름의 리스트]

열 순서 바꾸기
print(df.head(3))
# survived pclass sex age
# 0 0 3 male 22.0
# 1 1 1 female 38.0
# 2 1 3 female 26.0
columns = list(df.columns.values)
columns_reversed = list(reversed(columns))
df_reversed = df[columns_reversed]
print(df_reversed.head(3))
# age sex pclass survived
# 0 22.0 male 3 0
# 1 38.0 female 1 1
# 2 26.0 female 3 1

📚 리스트 요소 뒤집기: list객체.reverse()
📚 reversed된 객체 리턴: reversed(list/set/문자열 객체)

2-2. 열 분리#

📚 시리즈의 문자열 분리: Series객체.str.split()
📚 시리즈의 문자열 리스트 인덱싱: Series객체.str.get(인덱스)

열 분리하기
print(df.head(3))
# 연월일 거래량
# 0 2018-07-02 137977
# 1 2018-06-29 170253
# 2 2018-06-28 155769
df['연월일'] = df['연월일'].astype('str')
dates = df['연월일'].str.split('-')
print(dates.head(3))
# 0 [2018, 07, 02]
# 1 [2018, 06, 29]
# 2 [2018, 06, 28]
# Name: 연월일, dtype: object
df['연'] = dates.str.get(0)
df['월'] = dates.str.get(1)
df['일'] = dates.str.get(2)
print(df.head(3))
# 연월일 거래량 연 월 일
# 0 2018-07-02 137977 2018 07 02
# 1 2018-06-29 170253 2018 06 29
# 2 2018-06-28 155769 2018 06 28

df_boolean = titanic.loc[mask1 | mask2 | mask3]

isin() 필터링

df_isin = titanic[isin_filter]

4. 데이터프레임 합치기#

4-1. 데이터프레임 연결#

📚 데이터프레임 연결: pd.concat(데이터프레임의 리스트)

옵션설명
axis0: 행 방향(기본값)
1: 열 방향
join'outer': 두 데이터프레임의 합집합(기본값)
'inner': 두 데이터프레임의 교집합
ignore_index새로운 행 인덱스 설정 유무 (True/False(기본값))

축 방향은 기본 옵션으로 axis=0
열 이름에 대해서는 기본으로 join='outer'

데이터프레임 연결(concatenate)
df1 = pd.DataFrame({'a':['a0', 'a1', 'a2', 'a3'],
'b':['b0', 'b1', 'b2', 'b3'],
'c':['c0', 'c1', 'c2', 'c3']},
index=[0, 1, 2, 3])
df2 = pd.DataFrame({'a':['a2', 'a3', 'a4', 'a5'],
'b':['b2', 'b3', 'b4', 'b5'],
'c':['c2', 'c3', 'c4', 'c5'],
'd':['d2', 'd3', 'd4', 'd5']},
index=[2, 3, 4, 5])
result1 = pd.concat([df1, df2])
print(result1)
# a b c d
# 0 a0 b0 c0 NaN
# 1 a1 b1 c1 NaN
# 2 a2 b2 c2 NaN
# 3 a3 b3 c3 NaN
# 2 a2 b2 c2 d2
# 3 a3 b3 c3 d3
# 4 a4 b4 c4 d4
# 5 a5 b5 c5 d5
result2 = pd.concat([df1, df2], axis=1, join='inner')
print(result2)
# a b c a b c d
# 2 a2 b2 c2 a2 b2 c2 d2
# 3 a3 b3 c3 a3 b3 c3 d3

4-2. 데이터프레임 병합#

📚 데이터프레임 병합: pd.merge(df_left, df_right, how=, on=)

📚 IPython 디스플레이 설정 변경: pd.set_option(옵션)

옵션설명
'display.max_columns', n출력할 최대 열의 개수
'display.max_colwidth', n출력할 열의 너비
'display.unicode.east_asian_width', True유니코드 사용 너비 조정
