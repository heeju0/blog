테마 색 변경#

💡 https://v2.docusaurus.io/docs/styling-layout에서 Primary Color에 원하는 색의 헥스 색상 코드를 입력하여 코드를 가져오면 보다 편리하게 변경 가능

src > css > custom.css > :root 에서 작업

--ifm-color-primary: #3a66ad;
--ifm-color-primary-dark: #345c9c;
--ifm-color-primary-darker: #315793;
--ifm-color-primary-darkest: #294779;
--ifm-color-primary-light: #4070be;
--ifm-color-primary-lighter: #4877c2;
--ifm-color-primary-lightest: #628aca;

메인화면 컨텐츠 내용 변경#

💡 undraw.co: 테마 색으로 설정했던 --ifm-color-primary의 헥사 색상 코드를 입력하면 같은 색상의 일러스트들을 .svg 파일로 다운로드 가능
💡 unsplash.com: 이미지 다운 가능

src > pages > index.js > features 에서 제목, 이미지, 단락 수, 내용 변경 작업

const features = [
{
title: '제목',
imageUrl: '이미지 url',

<>

function Feature({imageUrl, title, description, secondParagraph}) {
const imgUrl = useBaseUrl(imageUrl);
return (
<div className={clsx('col col--4', styles.feature)}>
{imgUrl && (
<div className="text--center">
<img className={styles.featureImage} src={imgUrl} alt={title} />
</div>
)}
<h3>{title}</h3>
<p>{description}</p>
{/* <p>This is my text</p> */}
<p>{secondParagraph}</p>
</div>
);
}

메인화면 컨텐츠 줄 추가#

src > pages > index.js > Home 에서 작업
<main>과 </main> 사이에 다음 코드 복사 & 붙여넣기

{features && features.length > 0 && (
<section className={styles.features}>
<div className="container">
<div className="row">
{features.map((props, idx) => (
<Feature key={idx} {...props} />
))}
</div>
</div>
</section>
)}
