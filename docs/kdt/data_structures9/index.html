1. 정렬 (Sort))#

연습문제 - 가장 큰 수#

입출력 예#

numbersreturn
[6, 10, 2]"6210"
[3, 30, 34, 5, 9]"9534330"

강의 풀이#

  1. 빈 문자열로 수를 초기화한다.
  2. 수의 목록을 값을 크게 만드는 것 우선으로 정렬한다.
  3. 목록에서 하나씩 꺼내어 현재 수에 이어 붙인다.
크게 만드는 수의 기준
숫자의 길이가 같으면 큰 수를 선택한다.
숫자의 길이가 다르면 숫자를 반복해서 이어붙이고 그 값으로 크기를 비교한다.
ex) 34 (343434) vs. 343 (343343343)
def solution(numbers):
# O(n)
numbers = [str(x) for x in numbers]
# O(n log n)
numbers.sort(key=lambda x: (x*4)[:4], reverse=True)
if numbers[0] == '0':
# O(1)
answer = '0'
else:
# O(n)
answer = ''.join(numbers)
return answer

시간 복잡도: O(n log n)

내 풀이#

def solution(numbers):
number = sorted(map(str, numbers), key=lambda x: x*3, reverse=True)
return str(int(''.join(number)))
