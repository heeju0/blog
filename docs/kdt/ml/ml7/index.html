선형분류 (미완성)

💡 『KDT 프로그래머스 인공지능 데브코스』에서 공부중..👀

선형분류의 목표와 방법들#

분류(classification)의 목표#

입력벡터 x를 K개의 가능한 클래스 중에서 하나의 클래스로 할당하는 것

분류를 위한 결정이론#

확률적 모델 (probabilistic model)#

생성모델(generative model): p(x|Cₖ)와 p(Cₖ)를 모델링한 다음 베이즈 정리를 사용하여 클래스의 사후확률 p(Cₖ|x)를 구한다. 혹은 결합확률 p(x, Cₖ)를 직접 모델링할 수도 있다.

식별모델(discriminative model): p(Cₖ|x)를 직접 모델링한다.

판별함수 (discriminant function)#

확률값은 계산하지 않고 입력 x를 클래스로 할당받는 판별함수(discriminant function)를 찾는다. 여기서는 판별함수 중 선형함수만을 다룬다.

판별함수#

클래스가 두 개인 경우#

결정경계#

y(x) = 0일 때, (D - 1)차원의 hyperplane이 만들어진다. 이때, y(x) = 0을 만족하는 x의 집합을 결정경계(decision boundary)라고 한다.

w₀ < 0이면 결정경계면은 원점으로부터 w가 향하는 방향으로 멀어져 있으며, w₀ > 0이면 결정경계면은 원점으로부터 w의 반대 방향으로 멀어져 있다.

y(x)값의 부호의 의미#

y(x)값은 x와 결정경계면 사이의 부호화된 거리와 비례한다.

임의의 한 점 x의 결정경계면에 대한 사영을 x⊥라고 하면 다음 식이 성립한다.

ㅤㅤ

확률적 생성 모델 (Probabilistic Generative Models)#

분류문제를 확률적 관점에서 살펴본다. 선형회귀와 마찬가지로 확률적 모델은 통합적인 관점을 제공한다. 예를 들어, 데이터의 분포에 관해 어떤 가정을 두게 되면 앞에서 살펴본 선형적인 결정경계(linear decision boundary)가 그 결과로 유도되는 것을 보게 된다.

p(x|Cₖ)와 p(Cₖ)를 모델링한 후, 베이즈 정리를 사용하여 클래스의 사후확률 p(Cₖ|x)를 구한다. 이전의 판별함수 방법에서는 어떤 에러함수를 최소화시키기 위한 최적의 파라미터를 찾는 것이 목적이었다면, 확률적 모델은 데이터의 분포를 모델링하면서 분류문제를 결과적으로 풀게 된다.
