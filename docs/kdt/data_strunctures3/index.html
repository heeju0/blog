스택 (Stacks)#

자료(data element)를 보관할 수 있는 선형 구조

후입선출(LIFO, Last-In First-Out)의 특징을 가진다.
-> 스택에서는 데이터를 넣을 때 한 쪽 끝에서 밀어 넣어야 하며(push 연산), 꺼낼 때에는 같은 쪽에서 뽑아야 한다(pop 연산).

스택의 동작#

# 초기 상태: 비어 있는 스택
S = Stack()

초기 상태: 비어 있는 스택 (S = Stack())
데이터 원소 A를 스택에 추가 (S.push(A))
데이터 원소 B를 스택에 추가 (S.push(B))
데이터 원소 꺼내기 (r1 = S.pop(), r1에는 B가 담김) 데이터 원소 꺼내기 (r2 = S.pop(), r2에는 A가 담김)
비어 있는 스택에서 데이터 원소를 꺼내려 할 때 오류 발생 (r3 = S.pop())
