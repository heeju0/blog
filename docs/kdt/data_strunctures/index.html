1. 선형 배열 (Linear Arrays)#

데이터가 일렬로 늘어선 형태

1-1. 배열(Array)#

원소들을 순서대로 늘어놓은 것
파이썬에서는 리스트(List)를 사용하여 배열을 나타낼 수 있으며, 다른 언어들과는 다르게 데이터들이 같은 타입이 아니어도 입력 가능하다.

리스트 (배열) 연산#

📚 원소 덧붙이기: List객체.append(원소)
📚 끝에서 꺼내기: List객체.pop()
배열의 끝에 원소를 덧붙이는 연산과 끝의 원소를 꺼내는 연산은 리스트의 길이와 무관하게 시간 복잡도가 O(1)이다.

📚 원소 삽입하기: List객체.insert(idx, 원소)
📚 원소 삭제하기: del(List객체[idx])
리스트의 길이가 길수록 오래 걸리며, 시간 복잡도가 O(n)이다. 이와 같이 리스트의 길이에 비례한 시간을 선형 시간이라고 표현한다.

💡 del()과 pop()의 차이
pop(): 해당 원소를 꺼내는 개념. pop() 실행 시 해당 원소가 출력되며, 리스트에서는 해당 원소가 지워진다.
del(): 해당 원소를 제거하는 개념. del() 실행 시 출력값 없이 해당 원소가 지워진다.

📚 원소 탐색하기: List객체.index(원소)
시간 복잡도가 O(1)이다. (왜..?)
