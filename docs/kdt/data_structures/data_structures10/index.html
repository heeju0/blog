동적계획법

💡 『KDT 프로그래머스 인공지능 데브코스』에서 공부중..👀

동적계획법 (Dynamic Programming)#

주어진 최적화 문제를 재귀적인 방식으로 보다 작은 부분의 문제로 나누고 부분 문제를 차례로 풀어가면서 해를 조합하여 전체 문제의 해담에 이르는 방식

알고리즘의 진행에 따라 탐색해야 할 범위를 동적으로 결정함으로써 탐색 범위를 한정할 수 있다.

문제의 성질에 따라, 동적계획법으로 풀어냄으로써 탐색해야 하는 범위를 효과적으로 줄일 수 있다.

예제#

피보나치 수열을 재귀함수로 구현할 경우, 시간 복잡도는 지수 함수의 형태로 나타난다.
이것을 동적계획법으로 풀어내면,

f(0) = 0, f(1) = 1
f(2) = f(1) + f(0) = 1
f(3) = f(2) + f(1) = 2

동적계획법 적용의 예 - Knapsack Problem#

여러 물건들의 금액과 무게가 서로 다르고, 배낭에 넣을 수 있는 무게가 한정되어 있을 때, 가장 높은 금액을 가지도록 물건을 골라 담는다.

연습문제 - N으로 표현#

입출력 예#

Nnumberreturn
5124
2113

강의 풀이#

  1. N을 한 번 사용해서 만들 수 있는 수들을 찾는다.
  2. N을 두 번 사용해서 만들 수 있는 수들을 찾는다.
  3. 이 과정을 반복하면서 원하는 결과가 나오면 리턴한다.

일반화: 1과 n-1 사이의 사칙연산, 2와 n-2 사이의 사칙연산 등을 반복한다.

def solution(N, number):
s = [set() for _ in range(8)]
for i, x in enumerate(s, start=1):
x.add(int(str(N) * i))
for i in range(len(s)):
for j in range(i):
for op1 in s[j]:
for op2 in s[i - j - 1]:
s[i].add(op1 + op2)
s[i].add(op1 - op2)
s[i].add(op1 * op2)
if op2 != 0:
s[i].add(op1 // op2)
if number in s[i]:
answer = i + 1
break
else:
answer = -1
return answer
