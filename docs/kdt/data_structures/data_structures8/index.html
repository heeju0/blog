탐욕법 (Greedy Algorithm)#

알고리즘의 각 단계에서 그 순간에 최적이라고 생각되는 것을 선택하는 방법

현재의 선택이 마지막 해답의 최적성을 해치지 않을 때 탐욕법을 사용할 수 있다.

연습문제 - 체육복#

입출력 예#

nlostreservereturn
5[2, 4][1, 3, 5]5
5[2, 4][3]4
3[3][1]2

강의 풀이 1#

  1. 빌려줄 학생들을 정해진 순서로 살펴야 하고, 이 정해진 순서에 따라 우선하여 빌려줄 방향을 정해야 한다.
  2. 학생의 수는 기껏해야 30명이므로, 학생 수만큼 배열을 확보하고, 여기에 각자가 가지고 있는 체육복의 수를 기록한다.
def solution(n, lost, reserve):
u = [1] * (n + 2)
# 여벌을 가져온 학생 처리 - O(n)
for i in reserve:
u[i] += 1
# 체육복을 잃어버린 학생 처리 - O(n)
for i in lost:
u[i] -= 1
# 체육복 빌려주기 처리 - O(n)
for i in range(1, n+1):
if u[i-1] == 0 and u[i] == 2:
u[i-1:i+1] = [1, 1]
elif u[i] == 2 and u[i+1] == 0:
u[i:i+2] = [1, 1]
# 결과 리턴 - O(n)
return len([x for x in u[1:-1] if x > 0])

시간 복잡도: O(n)

강의 풀이 2#

(만약 전체 학생 수가 매우 크고, 여벌의 체육복을 가져온 학생은 매우 적다면?)

  1. reserve를 오름차순 정렬한다.
  2. reserve의 원소들을 차례로 살펴보면서 빌려줄 수 있는 다른 학생을 찾아 처리한다.
def solution(n, lost, reserve):
# 잃어버렸지만 여벌의 체육복이 있는 학생들 - set 연산은 전체의 길이에 비례
s = set(lost) & set(reserve) # 교집합
# 잃어버렸고 여벌의 체육복이 없는 학생들
l = set(lost) - s # 차집합
# 여벌의 체육복을 빌려줄 수 있는 학생들
r = set(reserve) - s
# 체육복 빌려주기 처리
for x in sorted(r):
if x - 1 in l:
l.remove(x-1)
elif x + 1 in l:
l.remove(x+1)
return n - len(l)

시간 복잡도:
여벌의 체육복을 가져온 학생의 수를 k라고 할 때, 정렬: O(k log k)
빌려줄 수 있는 다른 학생을 찾아 처리 (해시 적용): O(k)
따라서 시간 복잡도는 O(k log k)
-> n이 매우 크고 k << n일 때 유리함

내 풀이#

def solution(n, lost, reserve):
for i in lost:
if i in reserve:
lost.remove(i)
reserve.remove(i)
for i in sorted(reserve):
if i in lost:
lost.remove(i)
elif i-1 in lost:
lost.remove(i-1)
elif i+1 in lost:
lost.remove(i+1)
answer = n - len(lost)
return answer

연습문제 - 큰 수 만들기#

입출력 예#

numberreservereturn
"1924"2"94"
"1231234"3"3234"
"4177252841"4"775841"

강의 풀이#

원칙: 앞 자리에 큰 수가 오는 것이 전체를 크게 만든다.

  1. number로부터 하나씩 꺼내어 모으되, 이미 모아둔 것 중 지금 등장한 것보다 작은 것들은 빼낸다.
  2. k개만큼 제거하면 남은 숫자는 모두 담는다.
  3. 아직 k 만큼 채우지 못한 경우 뒤에서 남은 수만큼 빼낸다.
def solution(number, k):
collected = []
# O(n)
for i, num in enumerate(number):
# k가 남은 경우
while len(collected) > 0 and collected[-1] < num and k > 0:
collected.pop()
k -= 1
if k == 0:
collected += list(number[i:])
break
collected.append(num)
collected = collected[:-k] if k > 0 else collected
answer = ''.join(collected)
return answer

시간 복잡도: O(n)
