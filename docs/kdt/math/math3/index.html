통계학

💡 『KDT 프로그래머스 인공지능 데브코스』에서 공부중..👀

1. 개념 정의#

통계학 (Statistics)#

데이터의 수집(collection), 구성(organization), 분석(analysis), 해석(interpretation), 표현(presentation)에 관한 학문

크게 기술 통계학(desciptive statistics)과 추측 통계학(inferential statistics)로 구분할 수 있다.

모집단, 모수, 표본#

모집단 (Population)#

어떤 질문이나 실험을 위해 관심의 대상이 되는 개체나 사건의 집합
ex) 전교 남학생의 키

모수 (Parameter)#

모집단의 수치적인 특성
ex) 키의 평균

표본 (Sample)#

모집단에서 선택된 개체나 사건의 집합
ex) 전교생 중 30명만 뽑아서 조사

도수 (Frequency)#

어떤 사건이 실험이나 관찰로부터 발생한 횟수
표현 방법: 도수분포표(Frequency Distribution Table), 막대그래프 (Bar Graph, 질적 자료), 히스토그램 (Histogram, 양적 자료), 줄기-잎 그림(Stem and Leaf Diagram)

상대도수#

도수를 전체 원소의 수로 나눈 것
(확률을 계산할 때 중요한 개념이다!)

평균 (mean)#

모집단을 나타내는 가장 대표적인 모수

ㅤㅤ

a = [79, 54, 74, 62, 85, 55, 88, 85, 51, 85, 54, 84, 78, 47]
b = [79, 54, 74, 62, 85, 55, 88, 85, 51, 85, 54, 84, 78, 47, 1000]
len(a)
# 14
import statistics
statistics.mean(a)
# 70.07142857142857
statistics.mean(b)
# 132.06666666666666

평균은 극단 값의 영향을 많이 받는다.

모평균 μ#

모집단 전체 자료의 평균

표본 평균 x#

모집단에서 추출한 표본의 평균

중앙값 (Meadian)#

자료를 순서대로 나열했을 때 가운데 있는 값

statistics.median(a)
# 76.0
statistics.median(b)
# 78

분산 (Variance)#

편차의 제곱의 합을 자료의 수로 나눈 값

모분산 σ²#

자료가 모집단 경우

ㅤㅤ

statistics.pvariance(a)
numpy.var(a)
# 217.6377551020408

표본 분산 s²#

자료가 표본일 경우

ㅤㅤ

statistics.variance(a)
scipy.stats.tvar(a)
numpy.std(a, ddof=1)
# 234.37912087912088
statistics.variance(b)
# 57868.78095238096

b의 경우 이상값으로 인해 평균이 다른 값들과는 매우 떨어진 위치에 나오기 때문에 분산의 값이 커진다.

❗ ddof: Delta Degrees of Freedom

표준편차 (Standard Deviation)#

분산의 양의 제곱근

모표준편차 σ#

statistics.pstdev(a)
numpy.std(a)
# 14.752550799846134

표본표준편차 s#

statistics.stdev(a)
numpy.var(a, ddof=1)
# 15.30944547915178
statistics.stdev(b)
# 240.55930859640614

범위 (Range)#

자료를 정렬하였을 때 가장 큰 값과 가장 작은 값의 차이

극단 값이 존재하는 경우 범위만으로 전체 자료의 구조를 보기에 어려움이 있다. 이럴 때 사분위수가 유용하다.

사분위수 Quartile)#

전체 자료를 정렬했을 때, ¼, ½, ¾ 위치에 있는 숫자
Q1: 제 1 사분위수
Q3: 제 3 사분위수

numpy.quantile(a, .25)
# 54.25
numpy.quantile(a, .5)
# 76.0
numpy.quantile(a, .75)
# 84.75

사분위범위(IQR, Interquartile Range): Q3 - Q1

numpy.quantile(a, .75) - numpy.quantile(a, .25)
# 30.5

2. 확률 (Probability)#

확률의 정의#

상대도수에 의한 정의#

상대도수의 극한
ㅤㅤ*상대도수: 똑같은 실험을 무수히 많이 반복할 때 어떤 일이 일어나는 비율

고전적 정의#

표본 공간 (sample space)#

모든 가능한 실험 결과들의 집합

사건#

관심 있는 실험 결과들의 집합
표본 공간의 부분 집합

어떤 사건이 일어날 확률#

표본 공간의 모든 원소가 일어날 확률이 같은 경우 원소의 수에 비례한다.
확률 = 사건의 원소의 수 / 표본 공간의 원소의 수

확률의 계산#

고전적 확률#

표본 공간의 원소의 수, 사건의 원소의 수를 세야 하므로 경우의 수를 쉽게 셀 수 있는 방법이 필요하다.
따라서 조합을 계산할 필요가 있다.

조합 (Combination)#

어떤 집합에서 순서에 상관 없이 뽑은 원소의 집합
n개 중 r개를 뽑는 조합의 수:

ㅤㅤ

덧셈 법칙 (Addition Law)#

ㅤㅤ

예제#

1000명의 사람 중, 남자의 비율이 40%, 20세 미만의 비율이 43%, 20세 미만이면서 남자인 사람의 비율이 15%라고 한다. 임의로 한 명을 뽑을 때, 남자이거나 20세 미만일 확률을 구하여라.

P(남) = 0.4
P(<20) = 0.43
P(남 ⋂ <20) = 0.15
P(남 ∪ <20) = P(남) + P(<20) - P(남 ⋂ <20) = 0.68

서로 배반 (Mutually Exclusive)#

두 사건의 교집합이 공집합인 경우 서로 배반이라고 한다.

ㅤㅤ
ㅤㅤ

곱셈 법칙#

조건부 확률 (Conditional Probability)#

어떤 사건 A가 일어났을 때, 다른 사건 B가 일어날 확률

ㅤㅤ

예제#

주사위를 하나 던졌는데 4 이상의 수가 나왔을 때, 그 수가 짝수일 확률을 구하여라.

P(A ⋂ B) = 3/6 = 1/2
P(A) = 2/6 = 1/3
P(B|A) = P(A ⋂ B) / P(A) = 2/3

곱셈 법칙#

ㅤㅤ

예제#

어떤 학교의 학생 중 60%가 남학생이며, 남학생들 중 80%는 축구를 좋아한다. 임의로 한 명을 뽑았을 때, 축구를 좋아하는 남학생일 확률을 구하여라.

P(A): 남학생일 확률
P(B): 축구를 좋아할 확률
P(A) = 0.6
P(B|A) = 0.8 (남학생이면서 축구를 좋아할 확률)
P(A ⋂ B) = P(B|A)P(A) = 0.48 (남학생 중에 축구를 좋아할 확률)

서로 독립#

한 사건이 일어날 확률이 다른 사건이 일어날 확률에 영향을 미치지 않는 것

ㅤㅤ
ㅤㅤ

여사건#

사건 A가 일어나지 않을 사건
어떤 사건과 그 여사건은 서로 배반이다.

확률의 분할 법칙#

ㅤㅤ
ㅤㅤㅤㅤ

예제#

어떤 사파리의 동물 중 70%가 사자고, 나머지는 호랑이다. 사자의 60%가 2살 이상이며, 호랑이는 40% 정도가 2살 이상이다. 이때, 전체 동물 중 2살 이상인 동물의 비율은?

P(A): 사자일 확률
P(B): 2살 이상일 확률
P(A) = 0.7
P(B) = P(B|A)P(A) + P(B|A의 여집합)P(A의 여집합)
= 0.6*0.7 + 0.4*0.3 = 0.54

예제#

어떤 사파리의 동물 중 70%가 사자고, 나머지는 호랑이다. 사자의 60%가 2살 이상이며, 호랑이는 40% 정도가 2살 이상이다. 임의로 동물 한 마리를 선택했을 때 이 동물이 2살 이상이었다면, 이 동물이 사자일 확률을 구하여라.

P(A): 사자일 확률
P(B): 2살 이상일 확률
P(A ⋂ B) = P(B|A)P(A) = 0.6*0.7 = 0.42
P(B) = 0.6*0.7 + 0.4*0.3 = 0.54 (확률의 분할 법칙을 이용)
P(A|B) = P(A ⋂ B) / P(B) = 0.78

❗ 확률의 분할 법칙 예제에 몇 가지 정보가 추가됨으로써 확률이 조금 달라졌다.

베이즈 정리#

어떤 사건에 대해 사전 확률 원인에 대한 가능성과 사후 확률의 원인 가능성 사이의 관계를 설명하는 확률 이론

사전 확률(prior probability): 아무 조건이 없었을 때의 확률
사후 확률(posterior probability): 추가적인 정보를 습득함에 따라 조금 달라진 확률

ㅤㅤ

일반화된 정의#

표본 공간 S를 사건 B₁, B₂, ...로 분할한 상태에서 사건 A가 발생했을 때 Bₓ가 발생할 확률

ㅤㅤ

예제#

어떤 사람이 검은색과 흰색의 셔츠를 가지고 있는데, 매일 아침 3/4 정도는 검은색 셔츠를 입고, 1/4 정도는 흰색 셔츠를 입는다. 검은색 셔츠를 입었을 때 3/4 정도는 넥타이를 매고, 흰색 셔츠를 입었을 때는 1/2 정도 넥타이를 맨다고 하면, 어느 날 넥타이를 맸을 때 검은색 셔츠를 입었을 확률을 구하여라.

P(A): 검은색 셔츠를 입었을 확률
P(B): 넥타이를 맸을 확률
P(A ⋂ B) = 3/4*3/4 = 9/16
P(B) = 3/4*3/4 + 1/4*1/2 = 11/16
P(A|B) = P(A ⋂ B) / P(B) = (9/16)/(11/16) = 9/11
