note

seaborn document

1. Seaborn 시작하기#

Matplotlib을 기반으로 더 다양한 시각화 방법을 제공하는 파이썬의 라이브러리

seaborn 설치
pip install seaborn
seaborn 모듈 불러오기
import seaborn as sns

2. Seaborn Case Study#

s.fig.set_size_inches(): 플롯의 크기 설정 sns.kdeplot(): 커널 밀도 그림
ㅤㅤ*옵션
ㅤㅤshade: 곡선 아래 영역의 음영 여부 (True/False)
sns.countplot(): 카운트 그림
sns.catplot(x=, y=, hue=, data=): 캣 그림
ㅤㅤ*옵션
ㅤㅤkind: 데이터를 나타낼 그래프 형태 설정 (strip, violin, bar 등)
sns.stripplot(): 스트립 그림
sns.swarmplot(): 스왐 그림
sns.heatmap(): 히트맵

커널 밀도 그림 (Kernel Density Plot)#

x = np.arange(0, 22, 2)
y = np.random.randint(0, 20, 20)
sns.kdeplot(y, shade=True)
plt.show()

ㅤㅤ

카운트 그림 (Count Plot)#

범주형 컬럼의 빈도 수를 시각화하는 방법으로, groupby 후의 카운트하는 것과 동일한 효과가 있다.

vote_df = pd.DataFrame({'name':['Andy', 'Bob', 'Cat'],
'vote':[True, True, False]})
vote_df
# name vote
# 0 Andy True
# 1 Bob True
# 2 Cat False
sns.countplot(x=vote_df['vote'])
plt.show()

ㅤㅤ

cf> groupby를 사용하여 시각화
vote_count = vote_df.groupby('vote').count()
vote_count
# name
# vote
# False 1
# True 2
plt.bar(x=[False, True], height=vote_count['name'])
plt.show()

ㅤㅤ

캣 그림 (Cat Plot)#

숫자형 변수와 하나 이상의 범주형 변수의 관계를 보여준다.
여기서 캣은 연결을 의미하는 Concat에서 온 단어로, 복잡한 데이터에 적용하기에 유용하다.

covid = pd.read_csv('data/country_wise_latest.csv')
s = sns.catplot('WHO Region', 'Confirmed', data=covid)
s.fig.set_size_inches(10, 6)
plt.show()

ㅤㅤ

covid = pd.read_csv('data/country_wise_latest.csv')
s = sns.catplot(x='WHO Region', y='Confirmed', data=covid, kind='bar)
s.fig.set_size_inches(10, 6)
plt.show()

ㅤㅤ

스트립 그림 (Strip Plot)#

산점도와 유사하게 데이터의 수치를 표현한다.

sns.stripplot('WHO Region', 'Recovered', data=covid)
plt.show()

ㅤㅤ

스왐 그림 (Swarm Plot)#

스트립 그림과 거의 유사하지만, 점들이 겹치는 경우 점을 옆으로 분산하여 확인하기 쉽게 나타낸다.

sns.swarmplot('WHO Region', 'Recovered', data=covid)
plt.show()

ㅤㅤ

히트맵#

데이터의 행렬을 색상으로 표현하며, 상관계수를 나타낼 때 많이 사용한다.

sns.heatmap(covid.corr())
plt.show()
