Matplotlib 라이브러리

💡 『KDT 프로그래머스 인공지능 데브코스』에서 공부중..👀

note

matplotlib document

1. Matplotlib 시작하기#

파이썬의 데이터 시각화 라이브러리
matplotlib 라이브러리에서는 파이썬으로 plotting할 수 있는 pyplot 모듈을 가장 많이 사용한다.

matplotlib 설치
pip install matplotlib
matplotlib 모듈 불러오기
import matplotlib.pyplot as plt
matplotlib 라이브러리를 inline 환경에서(cell 안에서) 보이도록 설정
%matplotlib inline

2. 자주 사용되는 Plotting의 Options#

Matplotlib 기본 메서드#

plt.plot(데이터): plotting
ㅤㅤ*옵션
ㅤㅤlabel: 해당 플롯에 대한 라벨을 설정. 범례를 작성할 때 해당 라벨로 나타낼 수 있다.
plt.show(): plt를 확인하는 명령
plt.figure(figsize=): plotting 할 도면을 선언.
ㅤㅤ*옵션
ㅤㅤfigsize: 플롯의 크기 설정. 튜플 형태로 선언한다.
plt.xlabel(), plt.ylabel(): 각각 x축과 y축에 대한 설명 추가
plt.axis([x_min, x_max, y_min, y_max]): x축과 y축의 범위 설정
plt.xticks(), plt.yticks(): x축과 y축의 눈금 설정
plt.title(): 그래프에 제목 추가
plt.legend(): 그래프의 범례 추가

1차함수 그래프#

y = x + 1
plt.plot([1, 2, 3, 4, 5])
# plt.plot(x=index, y=[1, 2, 3, 4, 5])
plt.show()

ㅤㅤ

2차함수 그래프#

y = x²
x = np.array([1, 2, 3, 4, 5]) # 정의역
y = np.array([1, 4, 9, 16, 25]) # f(x)
plt.plot(x, y)
plt.show()

ㅤㅤ

데이터가 몇 개 없기 때문에 꺾이는 지점이 보인다. np.arange()를 이용하여 우리가 아는 2차함수 형태에 가까운 그래프를 그릴 수 있다.

x = np.arange(-10, 10, 0.01)
plt.plot(x, x**2, label='trend') # 선에 대한 라벨을 trend로 설정
plt.xlabel('x value')
plt.ylabel('f(x) value')
plt.axis([-5, 5, 0, 25])
plt.xticks([i for i in range(-5, 6)]) # -5에서 5까지 1 간격으로 설정
plt.yticks([i for i in range(0, 27, 3)])
plt.title('y = x² graph')
plt.legend()
plt.show()

ㅤㅤ

3. Matplotlib Case Study#

Matplotlib 그래프 메서드#

plt.plot(): 꺾은선 그래프
plt.scatter(): 산점도
plt.boxplot(): 박스 플롯
plt.bar(): 막대 그래프
plt.hist(): 히스토그램
ㅤㅤ*옵션
ㅤㅤbins: 계급의 개수나 계급이 나뉘는 지점 설정
plt.pie(): 원형 그래프
ㅤㅤ*옵션
ㅤㅤlabels: 각 부채꼴에 대한 라벨 추가

꺾은선 그래프 (Plot)#

이전 값에 비해 다음 값이 어떻게 변화하였는지 선으로 확인할 수 있기 때문에 시계열 분석에 많이 사용한다.

x = np.arange(20) # 0 ~ 19
y = np.random.randint(0, 20, 20)
plt.plot(x, y)
plt.axis([0, 20, 0, 20])
plt.yticks([0, 5, 10, 15, 20])
plt.show()

ㅤㅤ

산점도 (Scatter Plot)#

x와 y가 완전히 별개일 때 사용하며, 상관 관계 등을 확인할 수 있다.

plt.scatter(x, y)
plt.show()

ㅤㅤ

박스 플롯 (Box Plot)#

단일 변수에 대한 분포를 확인할 때 많이 사용한다.
수치형 데이터에 대한 정보(Q1, Q2, Q3, min, max)를 가지고 있다.

plt.boxplot((x, y))
plt.title('Box plot of x, y')
plt.show()

ㅤㅤ

막대 그래프 (Bar Plot)#

범주형 데이터의 값과 크기를 직사각형으로 나타낸다.

plt.bar(x, y)
plt.xticks(np.arange(0, 21, 1))
plt.show()

ㅤㅤ

히스토그램 (Histogram)#

도수분포를 직사각형의 막대 형태로 나타내며, 계급이 존재한다.
또한 히스토그램에서 범주는 연속적인 경우가 많기 때문에 그래프들이 이어져 있다.
막대 그래프의 x축은 데이터의 개별 변량이었다면, 히스토그램에서 x축은 변량을 범주화한 계급이다.

plt.hist(y, bins=np.arange(0, 21, 2))
xticks(np.arange(0, 21, 2))
plt.show()

ㅤㅤ

원형 그래프 (Pie Chart)#

데이터에서 전체에 대한 부분의 비율을 부채꼴로 나타낸며, 다른 그래프에 비해 비율을 확인하기에 용이하다.

z = [100, 300, 200, 400]
plt.pie(z, labels=['one', 'two', 'three', 'four'])
plt.show()
