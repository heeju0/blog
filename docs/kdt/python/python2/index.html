1. Pandas 시작하기#

pandas 설치
pip install pandas
pandas 모듈 불러오기
import pandas as pd

Series: 1-D labeled array
DataFrame: 2-D labeled table, Series의 병렬적인 모음

Prerequisite: Table#

행과 열을 이용하여 데이터를 저장하고 관리하는 자료 구조(컨테이너)
주로 행은 개체, 열은 속성을 나타낸다.

2. Pandas로 1차원 데이터 다루기 - Series#

Series 기본 메서드#

pd.Series(): Series객체 생성

s = pd.Series([1, 4, 8, 16, 26])
s
# 0 1
# 1 4
# 2 8
# 3 16
# 4 26
# dtype: int64
t = pd.Series({'one':1, 'two':2, 'three':3})
t
# one 1
# two 2
# three 3
# dtype: int64

Series객체.dtype: Series에 담긴 데이터의 타입을 확인할 수 있다.

s.dtype
# dtype('int64')

Series vs. ndarray#

Series는 ndarray와 유사하다.
인덱싱과 슬라이딩 방법이 동일하며, 부가적으로 조건을 이용하여 검색할 수 있다.

t[1]
# 2
s[s > s.median()] # s의 중앙값보다 큰 값들만 검색
# 3 16
# 4 26
# dtype: int64

또한 numpy의 메서드를 Series객체에 적용할 수 있다.

np.exp(s)
# 0 2.718282e+00
# 1 5.459815e+01
# 2 2.980958e+03
# 3 8.886111e+06
# 4 1.957296e+11
# dtype: float64

Series vs. Dictionary#

Series객체는 딕셔너리에서 key를 검색하고 값을 추가하는 방법과 유사한 방법을 적용할 수 있다.

t['one']
# 1
t['four'] = 4
t
# one 1
# two 2
# three 3
# four 4
# dtype: int64
'two' in t
# True
t.get('five') # 'five'가 있으면 값을 리턴하고, 없으면 아무것도 리턴하지 않음
t.get('five', 0) # 'five'가 있으면 값을 리턴하고, 없으면 0을 리턴

Series에 이름 붙이기#

Series객체는 name 속성을 가지고 있어 Series객체를 만들 때 이름을 붙일 수 있다.

s = pd.Series(np.random.randn(5), name='random_nums')
s
# 0 -0.756946
# 1 0.839884
# 2 0.410863
# 3 0.463816
# 4 -1.821163
# Name: random_nums, dtype: float64
s.name = '임의의 난수'
s
# 0 -0.756946
# 1 0.839884
# 2 0.410863
# 3 0.463816
# 4 -1.821163
# Name: 임의의 난수, dtype: float64

3. Pandas로 2차원 데이터 다루기 - DataFrame#

DataFame 기본 메서드#

pd.DataFrame(): DataFrame객체 생성

d = {'height':[1, 2, 3, 4],
'weight':[30, 40, 50, 60]}
df = pd.DataFrame(d)
print(df)
# height weight
# 0 1 30
# 1 2 40
# 2 3 50
# 3 4 60

DataFrame객체.dtypes: DataFrame에 담긴 데이터의 타입을 확인할 수 있다.

df.dtypes
# height int64
# weight int64
# dtype: object

pd.read_csv(csv파일 경로): csv 파일을 DataFrame객체로 생성한다.

covid = pd.read_csv('data/country_wise_latest.csv')

Pandas 활용#

일부분만 관찰하기#

DataFrame객체.head(n): 처음 n개의 데이터 참조 (기본값: 5)
DataFrame객체.tail(n): 마지막 n개의 데이터 참조 (기본값: 5)

데이터 접근하기#

DataFrame객체[컬럼명], DataFrame객체.컬럼명: 해당 컬럼의 값 가져오기

DataFrame객체의 각 컬럼은 Series객체이기 때문에 Series객체의 메서드를 그대로 사용할 수 있다.

covid['Confirmed'][1:5]
# 1 4880
# 2 27973
# 3 907
# 4 950
# Name: Confirmed, dtype: int64
예제) 신규 확진자가 100명이 넘는 국가
covid.loc[covid['New cases'] > 100, 'Country/Region'].head(3)
# 0 Afghanistan
# 1 Albania
# 2 Algeria
# Name: Country/Region, dtype: object
예제) WHO 지역(WHO Region)이 동남아시아인 국가
covid['WHO Region'].unique()
# array(['Eastern Mediterranean', 'Europe', 'Africa', 'Americas',
# 'Western Pacific', 'South-East Asia'], dtype=object)
covid.loc[covid['WHO Region'] == 'Eastern Mediterranean', 'Country/Region'].head(3)
# 0 Afghanistan
# 12 Bahrain
# 48 Djibouti
# Name: Country/Region, dtype: object

DataFrame객체.loc[row, col]: 인덱스를 이용하여 데이터를 가져온다.

books_dict = {'Available':[True, True, False],
'Location':[102, 215, 323],
'Genre':['Programming', 'Physics', 'Math']}
books_df = pd.DataFrame(books_dict,
index=['버그란 무엇인가', '두근두근 물리학', '미분해줘 홈즈'])
print(books_df)
# Available Location Genre
# 버그란 무엇인가 True 102 Programming
# 두근두근 물리학 True 215 Physics
# 미분해줘 홈즈 False 323 Math
books_df.loc['버그란 무엇인가']
# Available True
# Location 102
# Genre Programming
# Name: 버그란 무엇인가, dtype: object
예제) 책 '미분해줘 홈즈'는 대출이 가능할까?
books_df.loc['미분해줘 홈즈', 'Available']
# False

DataFrame객체.iloc[row인덱스, col인덱스]: 인덱스 번호를 이용하여 데이터를 가져오며, 슬라이싱도 가능하다.

books_df.iloc[1:, 0:2]
# Available Location
# 두근두근 물리학 True 215
# 미분해줘 홈즈 False 323

Groupby#

SQL의 groupby와 동일하다.
테이블을 Split, Apply, Combine 과정을 거쳐 더 좋은 결과를 얻는 것이 목표다.
Split: 특정한 기준을 바탕으로 DataFrame을 분할한다.
Apply: 일반적으로 sum(), mean(), median() 등의통계함수를 적용하여 각 데이터를 압축한다.
Combine: Apply된 결과를 바탕으로 새로운 Series를 생성한다. (group_key: applied_value 형태)

DataFrame객체.groupby(): groupby를 수행한다.

예제) WHO Region별 확진자 수
# Split
covid_by_region = covid['Confirmed'].groupby(by=covid['WHO Region'])
